## <u>Objectifs primaire de vraiProjet :</u>
- Vendre des produits :
    - Afficher les produits de <u>première qualité*</u> pas trop chère afin de les valoriser.
    - Intégrer un système de messagerie afin que les users puissent voir auprès de quel autre utilisateurs ils pourront prendre leurs produits.
    - Intégrer un système de paiement en ligne afin que les utilisateurs puissent payer directement sur le site.
    - Prix pas trop élevé

## <u>Critère pour un produit de première qualité*</u>:
- Pas du tout abîmé (qualité esthétique)
- Produit de très haute qualités (Frigo, TV, Meuble, Téléphone, Table, Meuble TV)
- Prix pas trop élevé
